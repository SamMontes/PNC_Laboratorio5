# Laboratorio N-Capas: API de Especímenes de Hyrule 🗡️🛡️

Este repositorio contiene la implementación de una API REST bajo la arquitectura N-Capas (Controller, Service, Repository, Entity) construida con Spring Boot, incluyendo estandarización de respuestas, paginación y manejo global de excepciones. Repositoria reutilizado para el Laboratorio #5.
